Docker的常用命令

帮助命令

镜像命令

  • docker images 查看所有本地的主机上的镜像
  • docker search 搜索镜像

这个指令来搜索和在网站仓库搜索是一样的

  • docker pull 下载镜像
  • docker rmi 删除镜像
  • docker save 保存镜像文件
  • docker load 加载镜像

容器命令

说明：有了镜像才可以创建容器

  • docker run 新建容器并启动
  • 退出容器
  • docker ps 查看运行的容器
  • docker rm 删除容器
  • 启动和停止容器的操作

常用其它命令

  • 后台启动容器
  • 查看日志
  • 查看容器中进程信息
  • 查看镜像的元数据
  • 进入当前正在运行的容器
  • 从容器内拷贝文件到主机上

Docker数据卷
数据卷(volume)一个虚拟目录，是容器内目录与宿主机目录之间映射的桥梁 命令 说明 docker volume create 创建数据卷 docker volume ls 查看所有数据卷 docker volume rm v_name 删除指定数据卷 docker volume inspect v_name 查某个数据卷的详情 docker volume prune 清楚数据卷 在执行docker run命令时，使用**-v 数据卷:容器内目录**可以完成数据卷挂载 当创建容器时，如果挂载了数据卷且数据卷不存在，会自动创建数据卷
